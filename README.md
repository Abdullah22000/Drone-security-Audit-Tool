# Drone-security-Audit-Tool
A GUI-based drone security audit tool developed using Python and Kali Linux to analyse network vulnerabilities in AR Drone 2.0 within a controlled and ethical testing environment.

<img width="1237" height="715" alt="image" src="https://github.com/user-attachments/assets/db14247b-62f6-491f-997e-cc65a44f46ce" />


This project presents a GUI-based Drone Security Audit Tool developed as part of an MSc Cyber Security dissertation. The tool is designed to analyse network-level vulnerabilities in the Parrot AR Drone 2.0 within a controlled lab environment.

Built using Python (Tkinter) on Kali Linux, the application integrates multiple security testing utilities into a single interface, enabling efficient and structured analysis of drone communication systems.

🔍 Key Features
Network connectivity testing (Ping)
Port scanning using Nmap
Wi-Fi reconnaissance and signal analysis
Live video stream access via FFplay
Passive packet capture using airodump-ng
Controlled deauthentication testing using aireplay-ng
🛠 Technologies Used
Python (Tkinter for GUI)
Kali Linux
Aircrack-ng
Nmap
FFmpeg
⚠️ Ethical Consideration

This tool is strictly developed for educational and research purposes only. All testing must be conducted in authorised environments with proper permission. The project focuses on security analysis rather than exploitation, aligning with academic and legal guidelines.
